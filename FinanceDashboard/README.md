#  Finance Dashboard Pro

A modern Python finance dashboard that:
- Fetches live stock or forex data using yfinance
- Displays interactive charts with matplotlib
- Saves results to CSV
- Features a sleek dark mode Tkinter GUI

 Tech Stack:
Python, pandas, matplotlib, yfinance, Tkinter

 Setup:
pip install -r requirements.txt

 Run:
python main.py

 Example Symbols:
AAPL | TSLA | BTC-USD | EURUSD=X
